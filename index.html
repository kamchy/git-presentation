Git w obrazkach.

Git w obrazkach to projekt, dzięki któremu prezentacja koncepcji związanych z tym systemem kontroli wersji stanie się trochę bardziej kolorowa. Jak wiadomo, obrazki przemawiają do wyobraźni znacznie mocniej, niż najmocniejsze słowa.

Chcesz poprawić? Sforkuj, napisz, skomentuj.

Masz pomysł na rozwinięcie tej prezentacji? Napisz komentarz, dodaj tzw. issue, napisz lub po prostu sforkuj projekt.
